# Resources

## Where to look for useful characters

A list of keywords that can be used either in character finder (macOS) or with some online search engines:

- Punc, Punctuation
- El, Ell, Ellipsis
- Syllabic
- loz, square, box, tri
- dis
- alch (for alchemy)
- APL (the APL language)
- Gen
- Quad (▜, ▞, ⁗, ◲, ◔)
- Lazy ∾
